# Sample
## plot 2d
```
sin(x)
```

## contour
```
sin(sqrt(x^2 + y^2) ) / sqrt(x^2 + y^2)
```

## 2d parametric function
```
sin(t), sin(2t)
```

## plot 3d
```
sin(sqrt(x^2 + y^2) ) / sqrt(x^2 + y^2)
```

## 3d parametric function
```
cos(t), sin(t), t
```
